## 🚀 Quick Start After Download

1. **Download everything** (folder size: ~1.2 GB)

2. **Install prerequisites** (One time)
   - Node.js: https://nodejs.org/
   - PostgreSQL: https://www.postgresql.org/download/

3. **Create database** (One time)
   ```bash
   psql -U postgres
   CREATE DATABASE paisaads;
   \q
   ```

4. **Run the project**
   ```bash
   cd pay
   ./start.sh        # Mac/Linux
   # OR
   start.bat        # Windows
   ```

5. **Open browser**
   - Frontend: http://localhost:3000
   - Backend: http://localhost:3002
